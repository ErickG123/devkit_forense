Rodar a API:
uvicorn src.api.main:app --reload

Criando o executável:
pip install pyinstaller

Build:
pyinstaller --onefile --name forensic-cli --console --add-data "src/core;core" --add-data "src/api;api" src/cli/cli.py

Tags e Releases:
git commit -m "[patch] Fix de bug" → incrementa patch automaticamente
git commit -m "[major] Quebra compatibilidade" → incrementa major
git commit -m "[minor] Adiciona nova feature" → incrementa minor

Instalar a lib localmente:
pip install -e .

Rodar APP (dentro de src):
python -m app.main
